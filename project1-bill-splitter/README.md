# Smart Bill Splitter 💰

A command-line tool to fairly split restaurant bills with tax and tip calculations.

## Features

- ✅ Calculate per-person cost
- ✅ Include tax and tip percentages
- ✅ Input validation (handles errors gracefully)
- ✅ Professional formatted output
- ✅ Easy to use interface

## How to Run

### Prerequisites
- Python 3.11 or higher

### Running the Program

1. Clone or download this repository
2. Navigate to project folder:
   ```bash
   cd project1-bill-splitter
   ```
3. Run the program:
   ```bash
   python bill_splitter.py
   ```
4. Follow the prompts

## Usage Example

```
========================================
        BILL SPLIT CALCULATOR
========================================

Enter total bill amount: $100
Enter tax percentage: 8
Enter tip percentage: 20
How many people: 4

========================================
           CALCULATION BREAKDOWN
========================================
Original Bill:     $100.00
Tax (8%):         $  8.00
Tip (20%):        $ 20.00
----------------------------------------
Total Amount:      $128.00
Number of People:  4
----------------------------------------
Each person pays:  $ 32.00
========================================
```
